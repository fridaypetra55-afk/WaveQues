# WaveQuest

WaveQuest is a learn-to-earn education protocol on Stellar (Soroban) that rewards learners with XLM for completing quizzes, maintaining streaks, and earning verifiable on-chain credentials.

This repository contains scaffolds for Soroban smart contracts, a NestJS backend, a Next.js 15 frontend, Docker configs, CI, and docs.

See subfolders for details.
# WaveQues
